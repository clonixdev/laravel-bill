# Laravel Bill
Billing package
### Install
Follow next steps:

```shell script
composer require clonixdev/laravel-bill
```


1. run composer require clonixdev/laravel-bill

2. run php artisan publish

3. run php artisan publish migrations

4. run php artisan migrate
